# Expense Tracker

## Description
This is a simple Python program that allows users to enter multiple expense amounts and calculates the total amount spent.

The program continuously accepts expense entries until the user types "quit". It also handles invalid inputs using exception handling.

## Features
- Add multiple expense transactions.
- Calculate and display the running total.
- Handle invalid inputs using ValueError.
- Use a sentinel value ("quit") to stop the program.
- Display the final total before exiting.

## Requirements
- Python 3.x

## How to Run

1. Save the program as `expense_tracker.py`.
2. Open a terminal or command prompt.
3. Navigate to the folder containing the file.
4. Run the program:

```bash
python expense_tracker.py
