# To-Do List Application

## Project Description

This project is a command-line To-Do List Application developed using Python. It allows users to add tasks, view stored tasks, and exit the application through a simple menu-driven interface.

## Objective

To understand and implement Python Lists, Dictionaries, Functions, Loops, and Enumerate while building a practical task management system.

## Features

* Add new tasks
* View all stored tasks
* Store tasks using dictionaries
* Manage multiple tasks using a list
* Menu-driven interface
* User-friendly interaction

## Technologies Used

* Python 3
* Lists
* Dictionaries
* Functions
* Enumerate
* Loops

## Project Structure

Functions Used:

1. add_task(task_name)

   * Creates a dictionary for the task.
   * Stores the task in a list.

2. view_tasks()

   * Displays all stored tasks.
   * Uses enumerate() for numbering.

3. main()

   * Controls the menu system.
   * Handles user interaction.

## How to Run

1. Install Python 3.
2. Save the program as `todo_list.py`.
3. Open Terminal or Command Prompt.
4. Run:

python todo_list.py
